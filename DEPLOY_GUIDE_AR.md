# طريقة رفع السيرفر مجاناً لحل مشكلة الحظر

بما أن شركة الاتصالات لديك تحجب سيرفرات Webull، سنقوم برفع الباك إند (Backend) إلى خدمة سحابية مجانية تدعى **Render**. هذه الخدمة ستعمل من أمريكا مباشرة، وبالتالي ستتجاوز الحظر تلقائياً.

### الخطوة 1: إنشاء حساب على GitHub (إذا لم يكن لديك)
1. اذهب إلى [github.com](https://github.com).
2. أنشئ حساباً جديداً (Sign Up).

### الخطوة 2: رفع ملفات الباك إند
(بما أن الكود موجود على جهازك، أسهل طريقة هي رفعه يدوياً أو عبر أداة GitHub Desktop، أو يمكنني مساعدتك إذا كنت تستخدم Git).

**الملفات المطلوبة هي التي داخل مجلد `backend` وهي:**
- `app.py`
- `requirements.txt`
- `Procfile`
- `webull_client.py`

### الخطوة 3: النشر على Render
1. ادخل على موقع [render.com](https://render.com) وأنشئ حساباً باستخدام GitHub الخاص بك.
2. اضغط على زر **New +** واختر **Web Service**.
3. اختر المستودع (Repository) الذي رفعت عليه الملفات في GitHub.
4. في صفحة الإعدادات:
   - **Name:** اختر أي اسم (مثلاً `peacemap-api`).
   - **Region:** اختر **US East** (مهم جداً ليكون في أمريكا).
   - **Runtime:** اختر **Python 3**.
   - **Build Command:** `pip install -r requirements.txt`
   - **Start Command:** `gunicorn app:app`
   - **Instance Type:** اختر **Free**.
5. اضغط **Create Web Service**.

### الخطوة 4: تحديث التطبيق
بمجرد اكتمال النشر، سيعطيك Render رابطاً جديداً يبدأ بـ `https://...onrender.com`.
1. انسخ هذا الرابط.
2. افتح تطبيق **Peace Map** على هاتفك.
3. اضغط على أيقونة الإعدادات.
4. ألصق الرابط الجديد في خانة **Server URL**.
5. اضغط **Save**.

الآن سيعمل التطبيق بدون الحاجة لتشغيل كمبيوترك وبدون VPN!
